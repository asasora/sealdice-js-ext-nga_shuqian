# sealdice-js-ext-nga_shuqian
## 简介 
搭载于Sealdice框架的JS扩展
用于收藏nga帖子，记录楼层后，在后续的记录中会生成跳转到所记录楼层的页。
总体来说应该会比nga自己的收藏好用点。
## 使用方法
在加载进框架前，先在js文件内添加你的cookie，否则可能导致访问nga出错。
在加载进框架后，使用```.书签```查看帮助信息。
```
<尖括号内为必填参数，写的时候不带尖括号>
[方括号内为选填参数，写的时候不带方括号]
# 后面为注释，解释命令的作用

.书签 记录 <楼层> [tid]
# 记录NGA帖子的tid和您的楼层，如果不填tid将默认修改收藏中的第一个。
# 注意楼层在前，tid在后，写反会导致记录错误。

.书签 我的
# 查看您所有的NGA书签。

.书签 清除 [tid]
# 将记录的帖子删除，不填tid将删除所有书签
```
